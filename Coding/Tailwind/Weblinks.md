Installationsanweisung mit Vite:
https://tailwindcss.com/docs/installation/using-vite

```
$ npm create vite@latest tailwind -- --template react
$ cd tailwind/
$ npm install
$ npm install tailwindcss @tailwindcss/vite
$ npm list tailwindcss
$ vi vite.config.js
--> import tailwindcss from '@tailwindcss/vite'

in CSS Datei:
--> @import "tailwindcss";

$ npm install postcss autoprefixer

$ npx tailwindcss-cli@latest init -p


Tool
Aufgabe
Tailwind CSS
Schreibt/generiert Utility-basiertes CSS
PostCSS
Führt CSS-Plugins aus (Framework drumherum)
Autoprefixer

Macht CSS browserkompatibel (fügt Präfixe hinzu)
```
